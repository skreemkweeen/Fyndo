# Fyndo Developer API

Welcome to the Fyndo Developer Documentation. Here you will find the public-facing REST endpoints for integrating third-party headless storefronts with Fyndo's backend.

## Headless Architecture

Fyndo allows premium brands to run their own custom web stores while utilizing our high-performance backend for inventory, messaging, and auction ledgers.
